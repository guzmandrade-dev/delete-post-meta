Delete Post Meta based on meta key. Wrapper and admin panel for `delete_metadata`.

== Description ==

Wrapper and admin panel for [delete_metadata](https://developer.wordpress.org/reference/functions/delete_metadata/).
You can select the object type you want to search to delete metadata. Default to `post`.

WARNING! Use this plugin with caution. It will delete all post meta based on a meta key.
